# KFMaster_Unity_SDK

# SDK版本2.0.1升级至2.0.2说明 #
- bug修复点：修复了在注销失败情况下无法隐藏悬浮窗口的bug
- 升级方法：直接替换KFMaster_Java_2.0.1.jar为KFMaster_Java_2.0.2.jar即可
- 其中KFMaster_Java_2.0.2.jar在libKFMaster_SDK_Untiy_2.0.2.unitypackage当中
